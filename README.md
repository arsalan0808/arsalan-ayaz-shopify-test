# Shopify Development Test

## 🔹 Overview
Custom landing page built using Shopify Liquid and vanilla JavaScript.

## 🔹 Shopify Implementation
- Custom section: `custom-banner-grid.liquid`
- Banner + Product Grid combined
- Dynamic product rendering
- Popup with product details
- Variant selection
- Add to cart functionality

## 🔹 Special Logic
- If user selects:
  - Color: Black
  - Size: Medium  
→ "Soft Winter Jacket" is automatically added to cart
